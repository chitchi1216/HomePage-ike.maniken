# Kentaro Concept Portfolio — HomePage

**真庭謙太郎**によるサステナビリティ・廃棄物処理・公的データ活用のコンセプトポートフォリオサイトです。  
GitHub Pages を使って静的HTMLとして公開しています。

## サイト概要

「資源と人を未来につなぐ」をテーマに、廃棄物処理と公的データを接続し、  
排出事業者・処理業者・地域社会の意思決定を高度化するためのコンセプト提案をまとめています。

## 公開コンテンツ

| ページ | 概要 |
|--------|------|
| [index.html](index.html) | トップページ。サイト全体のナビゲーション |
| [mashup_concept_v2.html](mashup_concept_v2.html) | **データマッシュアップ構想 v2** — 公的データ × 廃棄物処理の10サービスを体系化したメインページ |
| [mashup_infographic.html](mashup_infographic.html) | インフォグラフィック版。構想No.1〜10をビジュアルで整理 |
| [shiryo_item_add_edi.pdf](shiryo_item_add_edi.pdf) | 関連資料PDF |

## 使用データソース

- ColBase / 国立国会図書館デジタルコレクション (NDL)
- 国土数値情報 / e-Stat / J-STAGE
- 気象庁オープンデータ / J-PlatPat / e-Gov
- 基盤地図情報 / 文化遺産オンライン
